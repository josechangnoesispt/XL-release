Bem-vindo ao projeto de cálculo de 'Congruências'


Um SW com o qual você pode esquecer para sempre o cálculo tedioso de congruências.

Alguma teoria: congruências

Dados 3 números inteiros, a, b e c, diremos que "a é congruente com b módulo c se a diferença entre a e c for um múltiplo de b. Por outras palavras, o resto da divisão de a por b é c"

a é congruente com b módulo c se houver um número k tal que a-c = kb

Exemplos

Vejamos alguns exemplos para esclarecer o conceito

  • ¿10 é congruente com 3 módulo 4?, Sim, porque 10-4 é um múltiplo de 3

  • ¿11 é congruente com 4 módulo 3?, Sim, porque 11-3 é um múltiplo de 4

  • ¿11 é congruente com 4 módulo 6?, NÃO, porque 11-6 NÃO é um múltiplo de 4
